# PetZone

## Deskripsi
PetZone adalah website Klinik Hewan yang menyediakan informasi layanan kesehatan hewan, vaksinasi, grooming, operasi, dan artikel edukasi bagi pemilik hewan peliharaan.

## Fitur
- Halaman Beranda
- Informasi Layanan
- Artikel Edukasi
- Kontak Klinik
- Responsive Design

## Teknologi
- HTML5
- CSS3
